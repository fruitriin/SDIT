# 基本操作

## SDI ファーストの考え方

SDIT は「1ウィンドウ = 1セッション」がデフォルトです。

- 新しいウィンドウを開く → 新しいシェルセッションが起動
- ウィンドウを閉じる → そのセッションが終了
- タブバーは表示されない（1セッションしかないため）

複数セッションを束ねたいときだけ、縦タブ（セッションサイドバー）が出現します。

## ウィンドウ操作

| 操作 | macOS | Linux/Windows |
|---|---|---|
| 新しいウィンドウ | Cmd+N | Ctrl+Shift+N |
| ウィンドウを閉じる | Cmd+W | Ctrl+Shift+W |
| アプリケーション終了 | Cmd+Q | — |

### 閉じる確認

子プロセスが実行中のセッションを閉じようとすると、確認ダイアログが表示されます。
この挙動は `[window] confirm_close` で変更できます。

| 値 | 挙動 |
|---|---|
| `process_running`（デフォルト） | 子プロセス実行中のみ確認 |
| `always` | 常に確認 |
| `never` | 確認なし |

## セッションとサイドバー（縦タブ）

### セッションの追加

| 操作 | macOS | Linux/Windows |
|---|---|---|
| 新しいセッション（タブ）を追加 | Cmd+T | Ctrl+Shift+T |
| セッションサイドバーの表示/非表示 | Cmd+\\ | Ctrl+\\ |

セッションが2つ以上になると、セッションサイドバー（縦タブ）が自動的に表示されます。

### セッション切り替え

| 操作 | macOS | Linux/Windows |
|---|---|---|
| 次のセッション | Ctrl+Tab / Cmd+Shift+] | Ctrl+Tab |
| 前のセッション | Ctrl+Shift+Tab / Cmd+Shift+[ | Ctrl+Shift+Tab |

### セッションの切り出し

サイドバーのセッションを独立したウィンドウに切り出せます。

- **macOS**: Cmd+Shift+N（DetachSession）

セッション（PTY）は切れません。表示先だけが変わります。

## テキスト選択とクリップボード

### 選択

| 操作 | 動作 |
|---|---|
| マウスドラッグ | 範囲選択 |
| ダブルクリック | 単語選択 |
| トリプルクリック | 行全体を選択 |
| Cmd+A / Ctrl+A | 全テキストを選択 |

### コピー＆ペースト

| 操作 | macOS | Linux/Windows |
|---|---|---|
| コピー | Cmd+C | Ctrl+Shift+C |
| ペースト | Cmd+V | Ctrl+Shift+V |

複数行のテキストをペーストするとき、確認ダイアログが表示されます（`[paste] confirm_multiline = true`）。

### 選択時の自動コピー（オプトイン）

```toml
[selection]
save_to_clipboard = true
```

テキスト選択が完了すると自動的にクリップボードにコピーされます。

## 検索

| 操作 | macOS | Linux/Windows |
|---|---|---|
| 検索を開く | Cmd+F | Ctrl+F |
| 次の一致 | Cmd+G | Ctrl+G |
| 前の一致 | Cmd+Shift+G | Ctrl+Shift+G |

検索バーに入力すると、ターミナル出力内のすべての一致箇所がハイライト表示されます。

## URL クリック

ターミナル出力内の URL を **Cmd+クリック**（macOS）でブラウザで開けます。
URL にマウスをホバーすると下線が表示されます。

## スクロール

| 操作 | 動作 |
|---|---|
| マウスホイール | スクロール（1ノッチ = 3行、設定変更可） |
| キー入力 | 自動でボトムにスクロール |

スクロールバーはデフォルトで表示されます（`[scrollbar] enabled = true`）。

## フォントサイズ変更

| 操作 | macOS |
|---|---|
| 拡大 | Cmd+= / Cmd+Shift++ |
| 縮小 | Cmd+- |
| リセット | Cmd+0 |

## プロンプトジャンプ（シェルインテグレーション）

OSC 133 対応のシェルでは、プロンプト単位でジャンプできます。

| 操作 | macOS |
|---|---|
| 前のプロンプト | Cmd+↑ |
| 次のプロンプト | Cmd+↓ |

## IME 入力

日本語などのマルチバイト文字入力に対応しています。
IME の候補ウィンドウはカーソル位置に表示されます。

## セッション復元

SDIT は終了時にウィンドウの位置とセッション情報を保存します。
次回起動時に前回の状態を復元します。

```toml
[window]
restore_session = true  # デフォルト: true
```

## Quick Terminal（macOS）

グローバルホットキーで画面端からスライドインするターミナルです。

```toml
[quick_terminal]
enabled = true
position = "top"      # top / bottom / left / right
size = 0.4            # 画面比率（0.1-1.0）
hotkey = "ctrl+`"     # ホットキー
```

## コマンドパレット

登録されているすべてのアクションにアクセスできます。

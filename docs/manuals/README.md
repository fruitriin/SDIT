## 設定ファイルの場所

```
~/.config/sdit/config.toml
```

XDG_CONFIG_HOME が設定されている場合は `$XDG_CONFIG_HOME/sdit/config.toml` が使われます。

設定ファイルが存在しない場合、すべてデフォルト値で起動します。
設定ファイルの変更はホットリロードで即座に反映されます（一部の項目を除く）。

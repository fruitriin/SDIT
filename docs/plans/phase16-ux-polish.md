# Phase 16: UX・品質向上

**概要**: 日常使用で必要な UX 改善とセキュリティ機能を追加する。小〜中規模の独立タスクを集約。

各サブフェーズは以下:

| Phase | 内容 | 規模 |
|---|---|---|
| 16.1 | スクロールバー | 中 |
| 16.2 | 起動モード設定（Maximized/Fullscreen） | 小 |
| 16.3 | 閉じる前の確認ダイアログ | 中 |
| 16.4 | 選択色設定 | 小 |
| 16.5 | クリップボードコピー時末尾空白削除 | 小 |
| 16.6 | Working Directory 継承 | 小 |
| 16.7 | スクロールトゥボトム設定 | 小 |
| 16.8 | 最小コントラスト比 | 小 |
| 16.9 | Secure Keyboard Entry（macOS） | 小 |

## 実装順序の推奨

1. Phase 16.2（起動モード）— 最も低コスト
2. Phase 16.5（末尾空白削除）— 低コスト
3. Phase 16.4（選択色設定）— 低コスト
4. Phase 16.6（CWD 継承）— 低コスト
5. Phase 16.7（スクロールトゥボトム）— 低コスト
6. Phase 16.8（最小コントラスト比）— 低コスト
7. Phase 16.9（Secure Keyboard Entry）— 低コスト
8. Phase 16.1（スクロールバー）— 中規模
9. Phase 16.3（確認ダイアログ）— 中規模
